#### 3.字符串格式化

|     格式符号     |      转化    |
|----| ----|
|  %s |  将内容转为字符串 并将他放入占位符中 |
|  %d |  将内容转化为 数字类型 并将他放入占位符中 |
|  %f  |  将内容转化为 浮点数  并将他放入占位符中 |


```

```python
#  单个占位符
name = '张三'
message = '你好%s' % name

# 其中  %：表示我要占位  s：表示将字符串放入占位的地方
# 所以综合起来的意思就是 ：我先占个位等一会变量过来 我把它变成字符串放到占位的地方

# 多个占位符

# 多个变量占位要使用括号阔起来 并按照占位的顺序填入

year = 19
age = 20
message = '在20%s年 我%s岁了'%（year,age）

```

#### 4.格式化的精确控制
我们可以使用辅助符号m.n来控制数据的宽度和精度
m控制宽度 要求字数（很少使用）设置的宽度小于数字自身，不生效
.n 控制小数点精度 要求是数字 会进行四舍五入


> 示例:


%5d:表示将整数的宽度控制在5位，如数字11，被设置为5d，就会变成:空格空格空格11，用三个空格补足宽度。



%5.2f:表示将宽度控制为5，将小数点精度设置为2
小数点和小数部分也算入宽度计算。如，对11.345设置了%7.2f后结果是:空格空格11.35。2个空格补足宽度，小数部分限制2位精度后，四舍五入为.35

#### 5.字符串格式化方式2

>通过语法:f“内容变量”的格式来快速格式化

```python
name = '张三'
age = 20
money = 19.99
print(f'你好，我是{name}，今年{age}岁了，我的零花钱还剩{money}元')
```

这种方式适合场景
- 不用理会类型
- 不做精度控制

### 11.for循环

>区别 while 和for 循环的区别
>- while的循环是可以自定义的，自行控制循环条件
>- for循环是一种轮询机制，是对一堆内容进行逐一处理


> 语句：
> 		for 临时变量 in 待处理数据集：
> 				循环满足时执行的代码


ps：待处理数据集 严格成为序列类型   ，序列类型是指 其内容可以一个个的依次取出的一种类型 包括：字符串、列表、元组

#### 2.range  数字序列

>range（num）
获取一个从0开始到num结束的数字序列（不含num本身）
如range（4）  [0,1,2,3,]


>range（num1，num2）
获得一个从num1开始到num2结束的数字序列（不含num2本身）
如 range（1，3） [1,2]

>range(num1,num2,step)
>获得一个从num1开始到num2结束的数字序列（不含num2本身）
>数字之间的步长以step为准
>如 range（1，7，2） [1,3,5]
